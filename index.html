6 passos simples para colocar seu site no ar pelo github pages

  1. cadastre-se no github ou faça o seu login
  2. faça o uploud dos seus arquivos
  3. coloque o seu arquivo principal HTML em index.html. para o github indentificar o seu arquivo html principal
  4. após o uploud entre em no seu repositorio e vá em settings
  5. procure o nome pages do seu lado esquerdo
  6. vá em branch e onde tem o nome None coloque em main e click em Save do botão ao lado
